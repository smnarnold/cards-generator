# Cards Generator

## Requirements
- [NodeJS 6.x LTS](https://nodejs.org/en/)
- `npm install gulp-cli -g`
- `npm install yarn -g`

## Installing
    `npm i`

## Start Development
    `npm run start`
    
## Create Production build
    `npm run build`
    
## jQuery
- [jQuery 3.x](https://code.jquery.com/)
